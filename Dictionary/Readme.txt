Run Server:


Run this command to start server:
> java -jar server.jar [port]

Example: 
> java -jar server.jar 8000 

Tips:There is no need to input [dictionary-path] which has been included in the project. 


===========================================================================

Run Client


Run this command to start Client:
> java -jar client.jar [server-address] [server-port]

Example: 
> java -jar client.jar 127.0.0.1 8000
